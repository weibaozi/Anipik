# Anipik
**Features:**
- [x] RSS feed from anime sites
- [x] custom anime rules(keywords,episodes)
- [x] download through pikpak
- [x] discord notification 

**使用方法**

在setting.yaml里填写pikpak账号密码和下载路径:

location: 

pikpak_password: abc

pikpak_username: abc

之后run main和web

---
bot不用就不需要更改

填写token和你的频道channel后启动**run dcbot.bat**
